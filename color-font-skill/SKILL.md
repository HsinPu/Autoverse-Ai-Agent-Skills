# Color Font Skill

Use this skill to choose the palette and type direction first.

## Workflow

1. Decide the visual tone and the emotion the UI should communicate.
2. Choose a color palette with clear primary, secondary, neutral, and accent roles.
3. Pick a font pairing that fits the tone and preserves readability.
4. Check contrast, hierarchy, and usage consistency across the interface.
5. Hand the palette and typography direction to implementation skills.

## Rules

- Prefer a small, intentional palette over many competing colors.
- Make typography hierarchy obvious and consistent.
- Keep font choices readable on small screens.
- Align palette and type with the product and brand context.

## Handoff

- For implementation, use `frontend-design` or `design-consultation`.
- For system-level token work, use `design-system-patterns`.
