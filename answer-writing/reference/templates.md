# 常用模板（Templates）

以下模板以「可直接複製改寫」為目標。

## A) 教學/解釋型

- 先用 1–2 句講概念與適用範圍
- 給最小例子
- 給常見坑/注意事項

```text
概念：<一句話>
怎麼用：
- <步驟 1>
- <步驟 2>
範例：
<code>
注意：<1-2 點>
```

## B) Debug/排錯型

重點：讓使用者能「重現、定位、驗證」。

```text
推測原因：<最可能的 1-2 個>
你可以先做：
- <檢查 1>（預期看到什麼）
- <檢查 2>（預期看到什麼）
修正：
- <修正步驟>
驗證：
- <如何確認修好>
```

## C) 實作交付型（含檔案變更）

```text
我做了：<一句話>
變更點：
- <path>：<做了什麼、為什麼>
怎麼跑：
```bash
<commands>
```
預期結果：<成功的判斷>
```

## D) 決策建議型（選項/取捨）

```text
推薦：<我選 A，原因>
選項：
- A：<優點/缺點/適用>
- B：<優點/缺點/適用>
我會選 A 因為：<2-3 句>
如果你更在意 <X>，改選 B。
```

## E) 彙整/摘要型

```text
結論：<一句話>
重點：
- <1>
- <2>
下一步：
- <1>
```
