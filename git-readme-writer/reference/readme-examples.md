# README 範例索引（README Example Index）

## Purpose

這份索引用來導覽 `git-readme-writer` 的多檔 example packs。
先用 [readme-types.md](readme-types.md) 判斷 README 類型，再到對應範例檔查看完整 skeleton 與寫法說明。

## Example Packs

- **Minimal**：小型 repo、MVP、單一工具、剛起步專案，見 [readme-example-minimal.md](readme-example-minimal.md)
- **Product / App**：產品型、網站、dashboard、終端應用，見 [readme-example-product-app.md](readme-example-product-app.md)
- **Library / SDK**：套件、SDK、可整合模組，見 [readme-example-library-sdk.md](readme-example-library-sdk.md)
- **CLI Tool**：命令列工具、dev tool、ops utility，見 [readme-example-cli-tool.md](readme-example-cli-tool.md)
- **API Service**：HTTP API、GraphQL、gRPC、microservice，見 [readme-example-api-service.md](readme-example-api-service.md)
- **Template / Boilerplate**：starter、template repository、scaffold，見 [readme-example-template-boilerplate.md](readme-example-template-boilerplate.md)
- **Monorepo**：workspace root、多 package / app repo，見 [readme-example-monorepo.md](readme-example-monorepo.md)
- **Internal / Enterprise / Gerrit**：企業內部、權限控管、Gerrit review flow，見 [readme-example-internal-enterprise-gerrit.md](readme-example-internal-enterprise-gerrit.md)
- **Research / Demo**：論文實作、模型展示、prototype，見 [readme-example-research-demo.md](readme-example-research-demo.md)
- **Platform Overlays**：GitHub / Gerrit 專屬補充段落，見 [readme-example-platform-overlays.md](readme-example-platform-overlays.md)
- **Section Snippets**：可插拔的小段落，例如 badges、quick start、contributing、support，見 [readme-example-section-snippets.md](readme-example-section-snippets.md)

## How to Use

1. 先決定 primary README type。
2. 打開對應 example file，閱讀其 `When to Use`、`What to Highlight` 與 skeleton。
3. 若 repo 還需要平台差異，另外打開 `Platform Overlays`。
4. 若只想局部補強，再補 `Section Snippets`。
